# Churn Prediction Project

A machine learning project to predict customer churn using the Telco dataset.

## Models Used
- Logistic Regression
- Random Forest
- XGBoost

## Metrics Evaluated
- Accuracy
- ROC-AUC Score
- Classification Report

## Dataset
- Processed from Telco Churn Dataset (CSV)

## How to Run
1. Install dependencies:  
   `pip install -r requirements.txt`

2. Run model training script:  
   `python notebooks/phase4_model_training.py`
